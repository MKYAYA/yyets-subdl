# yyets-subdl
YYeTs (人人影视) 雙語字幕下載器，小小工具、大大方便！


## 一鍵下載字幕
* 給定關鍵字，自動尋找 YYeTs 相關字幕
* 下載完成、自動解壓縮，萃取「…繁體&英文.ass」


## 相依性
* Python [`requests`](http://docs.python-requests.org/en/latest/) 函式庫
* [`curl`](http://curl.haxx.se/) 工具


## 使用方式
* 找個地方放，例如 `/usr/local/bin`
* 命令列執行 `yyets-subdl <關鍵字>`
